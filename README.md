# dotfiles

My *nix dotfiles (managed by [chezmoi](https://www.chezmoi.io))

## Setup

- Install `chezmoi` following the instructions [here](https://www.chezmoi.io/install/)
- Run the following command and follow the prompts:
  ```bash
  chezmoi init --apply https://github.com/nikhil-prabhu/dotfiles
  ```
- Done! (you may need to restart your shell)

## Platform-specific fixes/tweaks

For platform-specific fixes/tweaks (Eg: bluetooth config, fstab options, etc.), check out the [Wiki](https://github.com/nikhil-prabhu/dotfiles/wiki)
